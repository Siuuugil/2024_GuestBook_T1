## 파일관리자
Win32 API의 파일 다이얼로그를 활용해 그림 데이터를 저장하고 불러오는 파일 관리 기능을 구현했습니다.
구현 내용

새 파일 생성 / 파일 저장 / 파일 불러오기 기능 구현
시스템 파일 탐색기 다이얼로그를 열어 저장 경로를 직접 지정할 수 있도록 처리
그림 데이터(펜 좌표, 색상, 두께, 시간 등)를 바이너리 파일로 직렬화하여 저장
저장된 파일 목록을 리스트박스 UI에 표시하고 선택 시 자동 로드
저장 파일 목록을 별도 파일로 유지해 프로그램 재실행 후에도 이전 파일 목록 복원
저장/불러오기 실패 시 예외 처리
